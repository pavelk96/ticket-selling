Ticket-selling

Проект для онлайн покупки билетов в кинотеатр, поиска информации о фильмах, сериалах.
Используемый стек при разработке:
- Frontend:
  - React
  - Redux
  - React router
  - UI library AntDesign
  - Webpack

- Backend:
  - NodeJS
  - Express
  - Jsonwebtoken

- База данных:
  - MongoDB
